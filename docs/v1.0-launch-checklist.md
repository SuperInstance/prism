## Emergency Procedures

### Critical Bug Found

1. **Assess severity**
   - Does it break core functionality?
   - Does it affect many users?
   - Is there a workaround?

2. **Communicate**
   - Post issue with "critical" label
   - Update README with known issue
   - Respond to affected users

3. **Fix**
   - Create fix branch
   - Write tests
   - Deploy fix
   - Publish patch version (1.0.1)

4. **Announce**
   - Post fix to GitHub
   - Update social media
   - Notify affected users

### Security Vulnerability Found

1. **Responsible disclosure**
   - Private report to maintainers
   - Don't publicize until fixed

2. **Fix and test**
   - Create security fix
   - Thorough testing
   - Security audit if needed

3. **Coordinate release**
   - Fix deployed
   - Advisory published
   - Users notified
